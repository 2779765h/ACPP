#### Application of DBSCAN Clustering to 4D-STEM Data

Student ID: 2779765h.
Applied computing in Physics Project. 

This is a package for the application of DBSCAN clustering on 4D-STEM data for identifying diffraction peaks and crystals using clustering results and Digital Dark Field images produced from the data. Functions include, DBSCAN and KMeans clustering, Digital Dark Field Images and computation of the centre of masses of clusters. Other functions can aid with ML and mathematics. There are also modified versions of the py4DSTEM Bragg Disk Detection Tutorial to provide examples of the process in different stages from Bragg Disk Detection, to saving Bragg Peak Data and progress towards Digital Dark Field Images.

#### Application of DBSCAN Clustering to 4D-STEM Data

Student ID: 2779765h.
Applied computing in Physics Project. 

This is a package for the application of DBSCAN clustering on 4D-STEM data for identitfying diffraction peaks and crystal using clustering results and Digital Dark Field Images produced from the data.  Functions include, DBSCAN and KMeans clustering, Digital Dark Field Images and clustering on the centre of Masses of clusters. Other functions can aid with ML and basic maths functions. There are also modified version of the py4DSTEM Bragg Disk Detection Tutorial to provide an example of the layout of the code.

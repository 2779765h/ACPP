This folder contains modified tutorials from py4DSTED tutorials, including basics_02_diskdetection for disk detection and basics_04_strain for centering. 
As well as notebooks containing the process for DBSCAN clustering and creating Digital Dark Field images.

Code is dependant on the open-source py4DSTEM toolkit  (https://github.com/py4dstem/py4DSTEM), where other specific functions or notebooks are referenced in the individiual notebooks in this folder.

#### Application of DBSCAN Clustering to 4D-STEM Data

Student ID: 2779765h.
Applied Computing in Physics Project. 

This is a package for the application of DBSCAN clustering on 4D-STEM data for identifying diffraction peaks and crystals using clustering results and Digital Dark Field images produced from the data. Functions include, DBSCAN and KMeans clustering, Digital Dark Field images and computation of the centre of masses of clusters. Other functions also include ML applications and plotting, and mathematics. There are also notebooks which describe the method in different stages from Bragg Disk Detection, to saving Bragg Peak Data and progress towards Digital Dark Field images as well as 1st and 2nd level clustering. Notebooks also contain code from the py4DSTEM tutorials as outlined in the notebooks.
